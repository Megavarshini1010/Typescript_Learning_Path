********** Day 01 - Introduction to TypeScript *********

## What is TypeScript?

TypeScript is an open-source programming language developed by Microsoft. It is a superset of JavaScript, meaning every valid JavaScript program is also valid TypeScript.

TypeScript adds static typing and modern development features that help developers write cleaner, safer, and more maintainable code.

## Why was TypeScript Created?

JavaScript is dynamically typed, so many errors are found only while the application is running.

TypeScript was introduced to solve these problems by providing compile-time type checking.

## Features of TypeScript

- Static Typing
- Compile-Time Error Checking
- Interfaces
- Generics
- Enums
- Type Inference
- Better IntelliSense
- Easier Refactoring

## Summary

TypeScript is JavaScript with static typing. It helps developers build reliable, maintainable, and scalable applications by catching errors during development instead of at runtime.
